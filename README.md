# Квиз Плиз

[Работающий продукт](https://praktika-1nca.onrender.com/)

## Что умеет приложение

- регистрация участников и организаторов;
- создание и редактирование квизов;
- текстовые вопросы и вопросы с изображением;
- один или несколько правильных ответов;
- подключение к игре по коду комнаты;
- проведение игры в реальном времени;
- подсчёт баллов и показ результатов;
- сохранение истории игр.

## Структура проекта

- `index.html` — главная HTML-страница;
- `src/app.js` — интерфейс приложения;
- `src/styles.css` — стили;
- `server/native.js` — сервер и игровая логика;
- `server/native-auth.js` — регистрация и авторизация;
- `server/store.js` — сохранение данных;
- `server/quiz-validation.js` — проверка квизов;
- `server/scoring.js` — подсчёт баллов.
